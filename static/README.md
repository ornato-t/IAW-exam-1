# Fogli di stile

Il file `custom.css` contiene le regole CSS personalizzate create ad hoc per il progetto.

I file `style.css` e `style.css.map` sono generati da Bootstrap e risultati dalla compilazione del file [`style.scss`](../assets/style.scss), contenuto nella cartella `../assets`.
